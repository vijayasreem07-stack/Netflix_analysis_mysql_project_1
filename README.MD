# Netflix Data Analysis Using MySQL

## Project Overview

This project explores Netflix's content catalog using MySQL. The objective is to analyze trends in movies, TV shows, ratings, countries, directors, genres, and Indian content available on Netflix.

## Objectives

* Analyze the distribution of Movies and TV Shows.
* Identify the top countries contributing content to Netflix.
* Explore content ratings and audience categories.
* Examine content growth over time.
* Discover top directors and genres.
* Analyze Indian content available on Netflix.
* Practice SQL concepts using a real-world dataset.

## Tools Used

* MySQL
* MySQL Workbench
* Netflix Titles Dataset

## SQL Concepts Applied

* SELECT
* WHERE
* GROUP BY
* ORDER BY
* LIMIT
* COUNT()
* LIKE
* CASE Statements
* String Functions

## Key Insights

* Movies constitute the majority of Netflix's content library.
* The United States contributes the highest number of titles.
* TV-MA is among the most common content ratings.
* Netflix experienced significant content growth in recent years.
* India contributes a substantial amount of content to the platform.
* Certain directors and genres dominate the catalog.
